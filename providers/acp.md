# ACP (Agent Client Protocol)

> Use Claude Code, Codex CLI, or Gemini CLI as LLM providers through the Agent Client Protocol.

🚧 **This page is under construction.** Content coming soon.

## Overview

ACP wraps command-line AI agents (Claude Code, OpenAI Codex CLI, Gemini CLI) as GoClaw providers. Instead of calling an HTTP API, GoClaw spawns the CLI as a subprocess and communicates via stdin/stdout.
