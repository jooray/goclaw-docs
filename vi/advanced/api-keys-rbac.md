# API Keys & RBAC

> Quản lý API key với phân quyền theo vai trò cho triển khai đa người dùng.

🚧 **Trang này đang được xây dựng.** Nội dung sẽ sớm được cập nhật.

## Tổng quan

GoClaw hỗ trợ xác thực bằng API key với ba vai trò RBAC: **admin** (toàn quyền), **operator** (đọc+ghi), và **viewer** (chỉ đọc). API key có thể tạo, liệt kê, và thu hồi qua REST API hoặc WebSocket RPC.
