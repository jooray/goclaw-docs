# ACP (Agent Client Protocol)

> Sử dụng Claude Code, Codex CLI, hoặc Gemini CLI làm LLM provider thông qua Agent Client Protocol.

🚧 **Trang này đang được xây dựng.** Nội dung sẽ sớm được cập nhật.

## Tổng quan

ACP bọc các AI agent dòng lệnh (Claude Code, OpenAI Codex CLI, Gemini CLI) thành GoClaw provider. Thay vì gọi HTTP API, GoClaw khởi chạy CLI như subprocess và giao tiếp qua stdin/stdout.
