# Channel Instances

> Run multiple accounts per channel type with dynamic instance management.

🚧 **This page is under construction.** Content coming soon.

## Overview

Channel Instances allow you to connect multiple accounts of the same channel type (e.g., multiple Telegram bots, multiple Slack workspaces). Each instance has its own credentials, writer permissions, and group configuration. Managed via REST API.
